# Tech Stack

- Framework: Next.js 16.2.10 (App Router)
- UI: React 19.2.4
- ภาษา: TypeScript
- Styling: Tailwind CSS v4 (ผ่าน `@tailwindcss/postcss`, ไม่มี tailwind.config)
- Font: Geist (โหลดผ่าน `next/font`)
- Linting: ESLint (flat config, `eslint.config.mjs`)
- Package manager: npm
- Testing: ยังไม่มี test runner
